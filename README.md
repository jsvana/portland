# Becoming Portland

A basic sidescrolling platformer (metroidvania).

## Dependencies

### OSX

Xcode (Tested with 6.1.1)

`brew install sdl2 sdl2_ttf sdl2_image cmake`

### Linux

Packages: `cmake`, `sdl2`, `sdl2_gfx`, `sdl2_image`, `sdl2_mixer`, `sdl2_ttf`, `smpeg2`

## Build and run

    $ ./scripts/run
